Installation

SXW.js is available for download as a package from npm and nuget. You can also download the plain javascript file from GitHub.

  • Installation
  • Basic usage
  • Customizations
  • Demo

Install using npm


You can install from npm using the following command
npm i @sxwjs/sxwjs

Install using nuget


SXW.js is also available as a nuget package. To install from nuget

  • Package Manger : Install-Package sxw.js
  • .Net Cli : dotnet add package sxw.js

Using js file directly


You can download the javascript file directly from GitHub and use it in your website.
